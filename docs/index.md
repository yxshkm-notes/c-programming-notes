## 📖 About This Documentation

This is a personal reference guide for **C Programming** and **Data Structures & Algorithms (DSA)**. It covers everything from the basics of C syntax to advanced data structures and algorithms — written in a clean, beginner-friendly format with real examples and code snippets.

Whether you're just starting out or revising for interviews, this doc has you covered. 🚀
